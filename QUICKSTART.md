# Quick Start Guide - GitHub Pages Version

## Getting Started in 3 Steps

### Step 1: Open the App
Visit: **https://mmcberry-bit.github.io/ai-chatbot-project/**

### Step 2: Get a GitHub Token
1. Go to [github.com/settings/tokens](https://github.com/settings/tokens)
2. Click **"Generate new token (classic)"**
3. Give it a name (e.g., "AI Chatbot")
4. No special scopes needed for public models
5. Click **"Generate token"**
6. Copy your token (starts with `ghp_`)

- Conversations are saved automatically

## Troubleshooting

**"Please configure your GitHub Token"**
- Click Settings and add your token

**"API request failed"**
- Check your token is valid
- Try a different AI model

**Chat not saving**
- Enable cookies/localStorage in your browser
- Don't use private/incognito mode if you want persistence
